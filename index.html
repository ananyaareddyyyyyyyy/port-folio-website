Recipe App

The Recipe App is a web application designed to help users discover, search for, and manage recipes. It features a home page that displays recipes in a grid or list format, with a search bar to filter recipes by name, ingredient, or category. Users can view detailed information about each recipe, including ingredients, instructions, cooking time, and images.

The Responsive Photo Gallery is a web application that showcases a collection of images in an organized and visually appealing manner. The gallery adapts seamlessly to different screen sizes and devices, ensuring a consistent and user-friendly experience whether on desktops, tablets, or smartphones.
